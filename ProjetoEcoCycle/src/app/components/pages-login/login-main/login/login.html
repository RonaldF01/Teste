Política de Privacidade

Última atualização: 11 de julho de 2026

A EcoCycle respeita a privacidade de seus usuários e reconhece a importância da proteção dos dados pessoais. Esta Política de Privacidade explica como os dados poderão ser coletados, utilizados, armazenados e protegidos durante a utilização da plataforma.

A EcoCycle é um protótipo acadêmico desenvolvido com foco em frontend. Os dados informados são armazenados localmente no navegador do usuário por meio do localStorage e não são enviados para um servidor ou banco de dados central.

1. Dados tratados

A plataforma poderá armazenar os seguintes dados fornecidos diretamente pelo usuário:

  • e-mail;
  • senha utilizada no protótipo;
  • tipo de perfil;
  • nome da organização;
  • CNPJ;
  • telefone;
  • endereço, cidade e estado;
  • dados de solicitações de coleta;
  • tipo e peso estimado dos resíduos;
  • data, horário e endereço da coleta;
  • observações fornecidas pelo usuário.

2. Finalidade dos dados

Os dados são utilizados para:

  • criar e autenticar usuários;
  • identificar o tipo de perfil;
  • exibir os dados da organização;
  • registrar solicitações de coleta;
  • exibir coletas disponíveis às cooperativas;
  • associar coletas aos usuários envolvidos;
  • demonstrar as funcionalidades da plataforma;
  • manter a sessão do usuário no navegador.

3. Armazenamento local

As informações ficam armazenadas no localStorage do navegador. Isso significa que os dados permanecem apenas no navegador e no dispositivo em que foram cadastrados.

  • outro computador não terá acesso automático aos mesmos dados;
  • limpar os dados de navegação poderá excluir os cadastros;
  • a aplicação não sincroniza dados entre dispositivos;
  • o modo anônimo poderá apagar os dados ao encerrar a sessão.

4. Compartilhamento

Os dados armazenados no navegador não são enviados automaticamente ao Vercel, GitHub ou a terceiros pela lógica deste protótipo.

Algumas informações operacionais podem ser exibidas entre os perfis simulados dentro do mesmo navegador. Por exemplo, uma coleta cadastrada por um Gerador poderá aparecer na área Match & Mapa de uma Cooperativa.

5. Segurança

O localStorage é utilizado apenas para fins acadêmicos e de demonstração. Ele não é adequado para armazenar senhas ou dados pessoais em uma aplicação comercial real.

Utilize dados fictícios durante testes e apresentações.

6. Retenção e exclusão

Os dados permanecerão no navegador até que sejam removidos pela aplicação, apagados manualmente ou excluídos por meio das configurações do navegador.

7. Direitos do titular

O usuário poderá solicitar ou realizar, quando aplicável:

  • acesso aos dados;
  • correção de informações;
  • exclusão dos dados;
  • informações sobre o tratamento;
  • revogação do consentimento, quando aplicável.

8. Dados de crianças e adolescentes

A plataforma não é destinada a crianças e adolescentes e não busca coletar conscientemente dados pessoais desse público.

9. Alterações

Esta política poderá ser atualizada para refletir alterações na plataforma, nas funcionalidades ou na legislação aplicável.

10. Contato

Equipe EcoCycle contato@ecocycle.com.br

Termos de Uso

Última atualização: 11 de julho de 2026

Estes Termos de Uso estabelecem as condições de acesso e utilização da plataforma EcoCycle.

Ao utilizar a aplicação, o usuário declara ter lido e compreendido estas condições.

1. Natureza da plataforma

A EcoCycle é um protótipo acadêmico desenvolvido para demonstrar uma solução frontend voltada à gestão e rastreabilidade de resíduos.

Esta versão não representa um serviço ambiental, logístico, fiscal, jurídico ou comercial real.

2. Perfis disponíveis

  • Gerador: pode solicitar coletas e acompanhar funcionalidades relacionadas à geração de resíduos.
  • Cooperativa: pode visualizar e aceitar coletas disponíveis.
  • Recicladora: pode utilizar as funcionalidades de recebimento e controle de volume.

3. Cadastro

O usuário é responsável por fornecer informações corretas e atualizadas.

Não é permitido:

  • utilizar dados de terceiros sem autorização;
  • tentar acessar a conta de outro usuário;
  • inserir conteúdo ilegal ou fraudulento;
  • explorar falhas da aplicação;
  • comprometer a segurança do sistema.

4. Senha

O usuário é responsável por manter a confidencialidade de sua senha.

Como esta versão armazena credenciais localmente no navegador, não devem ser utilizadas senhas reais ou reutilizadas em outros serviços.

Não utilize senhas bancárias, profissionais ou pessoais reais.

5. Solicitações de coleta

As solicitações cadastradas têm finalidade demonstrativa. O registro ou aceite de uma coleta não cria obrigação contratual real entre os usuários.

6. Uso permitido

A plataforma deve ser utilizada para finalidades lícitas, acadêmicas, educacionais e demonstrativas.

7. Uso proibido

  • utilizar a aplicação para atividades ilegais;
  • introduzir códigos maliciosos;
  • alterar indevidamente dados de outros usuários;
  • usar a plataforma como comprovante ambiental oficial;

8. Disponibilidade

Não existe garantia de funcionamento contínuo ou sem erros. A aplicação poderá ficar indisponível por manutenção, falhas de hospedagem ou problemas no dispositivo do usuário.

9. Limitação de responsabilidade

Os desenvolvedores não se responsabilizam por:

  • perda de dados armazenados no navegador;
  • uso de dados reais inseridos pelo usuário;
  • uso da aplicação fora de sua finalidade acadêmica;
  • falhas causadas pelo navegador ou dispositivo;
  • compartilhamento indevido de credenciais.

10. Propriedade intelectual

A identidade visual, o código, os textos, as telas e demais elementos da EcoCycle pertencem aos respectivos responsáveis, respeitadas as licenças de conteúdos de terceiros utilizados no projeto.

11. Privacidade

O tratamento dos dados está descrito na Política de Privacidade disponível neste mesmo modal.

12. Legislação aplicável

Estes termos são interpretados conforme a legislação brasileira, observada a natureza acadêmica do protótipo.
